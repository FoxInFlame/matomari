anime/top is a method to get the top anime on MyAnimeList.
GET anime/top
Method GET
Authentication None Required
Parameters
Name Possible Values Description
sort [Optional] all, airing, upcoming, tv, movie, ova, special, bypopularity, favorite
(default: all)		 Sort the top anime. All of these values are the same name as the parameters used in https://myanimelist.net/topanime.php.
page [Optional] Any natural number.
(default: 1)		 Page number for the top anime. If the page doesn't exist, it will become 1.
details [Optional] true or anything else for false
(default: none)		 Setting this to true will return the synopsis snippet, genres, release year, member count, score count and all that but the request will take more than 50 seconds. Not recommended for use.
Status Codes 200 (OK)
404 (MyAnimeList is offline or their code changed)

See the message section in the error response for more details about an error.
