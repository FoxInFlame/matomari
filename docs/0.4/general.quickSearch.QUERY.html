general/quickSearch/:query is a method to get 10 anime search results of a query on MAL.
GET general/quickSearch/:query
Method GET
Authentication None Required
Parameters
Name Possible Values Description
:query Search query. It must be at least 3 letters long if specified. If unspecified, a filter parameter needs to be specified instead. Show anime that's related to the specified query.
filter [Optional] Filters, separated by a comma
(default: none)		 Filters will limit the results and show only what you need.
Status Codes 200 (OK)
400 (Query must be at least 3 letters long)
404 (MAL is offline)
500 (The code for MAL is not valid HTML markup)

Filters

Filters allow you to limit the amount of results that will be returned. They should be in the query parameter, and each filter should be separated by a comma.
The available filters are:
Filter Description
all Do not Limit your search result
anime Limtis your search result to animes
manga Limtis your search result to mangas
person Limtis your search result to persons
character Limtis your search result to characters
news Limtis your search result to news
featured Limtis your search result to featured
forum Limtis your search result to forum topics
club Limtis your search result to clubs
user Limtis your search result to users
Example Filters:
/quickSearch/?q=Pokemon?filter=anime
/quickSearch/?q=Naruto?filter=anime,manga
