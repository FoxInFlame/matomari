anime/search/:query is a method to get anime search results of a query on MAL.
GET anime/search/:query
Method GET
Authentication None Required
Parameters
Name Possible Values Description
:query [Optional] Search query. It must be at least 3 letters long if specified. If unspecified, a filter parameter needs to be specified instead. Show anime that's related to the specified query.
page [Optional] Any natural number.
(default: "1")		 Page number for the top anime. If the page doesn't exist, it will become 1.
filter [Optional] Filters, separated by a comma
(default: none)		 Filters will limit the amount of anime and show only what you need.
Status Codes 200 (OK)
400 (Query must be at least 3 letters long)
404 (MyAnimeList is offline or their code changed)

See the message section in the error response for more details about an error.

Filters

Filters allow you to limit the amount of anime that will be returned. They should be in the query parameter, and each filter should be separated by a comma.
The available filters are:
Filter Possible Values Description
type TV, OVA, Movie, Special, ONA, Music Fairly obvious. Limits your search result to the specific type. The last specified one is used when it is declared multiple times.
score 1 - 10 Limtis your search result to anime with a community score starting with the specific number. It can be declared multiple times.
status finishedairing, currentlyairing, notyetaired Limits your search result to anime with the specific status. They're pretty much self-explanatory. The last specified one is used when it is declared multiple times.
producer TBC
Too much to handle right now, since I would have to list and sort all the hundreds of producers on MyAnimeList.
rating G, PG, PG13, R, R+, RX Obvious here as well. Remember to encode the plus sign if you're going to filter with R+ (+ becomes %2B). The last specified one is used when it is declared multiple times.
startdate YYYYMMDD format number. Use hyphens to leave a part blank. Limits your search result to anime with specific starting date, month, year, or a combination. It can be declared multiple times.
enddate YYYYMMDD format number. Use hyphens to leave a part blank. Limits your search result to anime with specific ending date, month, year, or a combination. It can be declared multiple times.
startswithletter Anything in the English alphabet, excluding numbers and special characters. Limits your search result to anime that has a title starting with a specific letter. The last specified one is used when it is declared multiple times.
inc-genre Any genre on MAL. Limits your search result to anime with a specific genre. It can be declared multiple times.
exc-genre Any genre on MAL Limits your search result to anime without the specific genre. It can be declared multiple times.
Example Filters:
/search/Pokemon?filter=inc-genre:action,score:9
/search/Naruto?filter=type:tv,rating:pg
/search/?filter=inc-genre:action
