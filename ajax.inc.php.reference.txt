This is a reference for all the methods available for ajax.inc.php on MyAnimeList.

https://myanimelist.net/includes/ajax.inc.php?t=[NUMBER HERE]

As you can probably see (16 comes after 1) there are tons of methods missing.
Please contribute to this by i-don't-know-how-the-hell-people-can-corperate-on-a-single-document and add the methods you know.

NUBMER:
  METHOD
  QUICK DESCRIPTION
  Parameter (GET) / Data to be sent (POST)
  Example usage (How it's used in MAL)

1:
  POST
  Something to do with hiding topics in the forums
  Data: {
    val:n,
    id:e,
    timestamp:f
  }
  Success: function(e){
    if(e==1) {
      t.html("Hid Topic");
    } else {
      if(e==2) {
        t.html("Removed Hide");
      } else {
        t.html("Too Many");
      }
    }
  }
  
16:
  POST
  Ignore Board
  Data: {
    val: t, (if t= 2, show Ignore Board, else show Un-Ignore Board)
    ibid: e,
    timestamp:f
  }
  Success: function(e){
    if(t==2) {
      n.data("mode",1),
      n.html("Ignore Board")
    } else {
      n.data("mode",2);
      n.html("Un-ignore Board")
    }
  }

69:
  POST
  Something to do with whether you are watching a topic or not
  Data: {
    topic_id:e,
    timestamp:f
  }
  Success: function(data){
    if(t) {
      n.html(data);
    } else {
      var o=i.search(/Watching/);
      o!=-1 ? d("#wt" + e).html('<img src="/images/watch_y.gif" title="You are watching this topic">') : d("#wt"+e).html('<img src="/images/watch_n.gif" title="You are not watching this topic">');
    }
  }

82:
  var e = d("#topicId").val();
  d.ajax({
    type:"POST",
    url:"/includes/ajax.inc.php?t=82",
    data:"topicId="+e+"&messageText="+encodeURIComponent(d("#messageText").val())+"&totalReplies="+d("#totalReplies").val(),
    success:function(t){
      if(null==t.html) {
        if(null!=t.errors) {
          alert(t.errors[0].message);
        } else {
          alert("There was an error posting, please try again.");
        }
      } else{
        var n="";
        if(t.moreposts){
          var a="";
          if(t.moreposts>1) {
            a = "s have"
          } else {
            a = " has";
            n = '<div class="goodresult">'+t.moreposts+" post"+a+' been added since your last page refresh. <a href="?topicid='+e+'&goto=newpost">Go to new post</a>.</div>'
          }
          d(".forum_boardrowspacer").last().after(n+t.html);
          d("#postEditButtons"+t.msg_id+" .quickEdit").click(i);
          d("#messageEdit"+t.msg_id+" .postEditedMessage").click(o);
          d("#messageEdit"+t.msg_id+" .clearEditedMessage").click(r);
          d("#messageText").val(""),d("#quickReply").hide();
          load_img_tags();
        }
        d("#postIndicator").hide()
      }
    },
    dataType:"json"
  });
  
84:
  POST
  Delete a forum message
  n is probably the post ID,t is the parent of ("div.postActions a.yes")
  Data: {
    "msgId=" + n
  }
  DataType: "json",
  Success: function(e) {
    if(e.error == null) {
      t.html("Deleted");
      d("#forumMsg" + n).hide("slow");
    } else {
      alert("Error: " + e.error);
    }
  }
  
85:
  POST
  Something to do with messages (Probably get the message text for editing an existing message)
  Data: {
    msg_id:e
  }
  Success: function(e){
    (n is probably the forum message element, i is idk, and o is the editor)
    e && (
      n.hide(),
      i.show(),
      o.val(e.message),
    )
  }

86:
  POST
  Something to do with messages (Probably editing an existing message)
  Data: {
    msg_id:e
    msg:a
    topic_id:r
  }
  Success: function(e){
    (t is probably the forum message element, n is the editor, and i is idk)
    e && (
      if(e.error) {
        alert(e.error_message);
        n.hide();
        i.show();
      } else {
        n.hide();
        i.hide();
        t.html(e.message_html);
        t.show();
      }
    )
  }
